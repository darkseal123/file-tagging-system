# file-tagging-system
A small tool to tag files and folders, convert word and ppt to pdf
first, tags are grouped into groups.
second, under each group, you can setup tags.
third, there is a input-box to use your manual tag.
fourth, there are time stamp.
fifth, there are tools to convert ppt and word to pdf.
sixth, there is a tool to combine pdf together.
seventh, the py reads the tags from the txt so change the txt changes the tags.
